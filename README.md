## 🧩 محتويات المشروع

### 📱 [Khassab App](./Khassab%20App/)
<img width="1835" height="1023" alt="Image" src="https://github.com/user-attachments/assets/6766acbe-b84c-4013-ac54-20791eb251da" />
تطبيق Flutter بيئي تفاعلي للمستخدمين، مرتبط بالعربات الذكية.

**الميزات:**
- واجهة عربية (RTL) باستخدام Material Design 3
- تسجيل دخول مع Firebase Authentication
- خرائط Google Maps لتحديد أقرب عربة
- مسح QR Code بعد التسميد
- نقاط بيئية وإنجازات للمستخدم
- نظام تتبع للشجرة الافتراضية
- لوحة دعم ودردشة بيئية

---

### 🔧 [khassab_smart_cart.ino](./khassab_smart_cart.ino)
كود Arduino المتكامل لتشغيل العربة الذكية الخاصة بالمشروع، يشمل:
- حساس وزن (HX711)
- حساس مسافة
- محرك غطاء (Servo)
- وحدة GPS لتحديد موقع التسميد

---

### 🤖 [khassab_model.ipynb](./khassab_model.ipynb)
نموذج تعلم آلي (ML Model) لتحليل البيانات البيئية أو التنبؤ بكفاءة التسميد، قابل للتطوير.

---

## 🎯 رؤية المشروع

مشروع خصب يهدف إلى حماية البيئة وتعزيز الاستدامة عبر إعادة تدوير النفايات العضوية ميدانيًا وتحفيز الزوار على المشاركة.

### ♻️ الأهداف:
- تقليل النفايات العضوية عبر تحويلها لسماد
- رفع الوعي البيئي للسكان والزوار
- تحفيز السلوك الإيجابي بنظام نقاط ومكافآت
- تتبع وتحليل الأثر البيئي للفرد والموقع
